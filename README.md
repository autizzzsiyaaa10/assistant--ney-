# 🌸 ney // Advanced Pastel AI Assistant

Welcome to **ney**, an enterprise-grade, pastel-themed conversational companion built with Python, Streamlit, and the Gemini AI engine.

## ✨ Features
* **Adaptive Themes:** Automatically shifts between soft light and dark pastel modes.
* **Hyperparameter Control:** Real-time AI temperature/creativity adjustment sidebar.
* **State Management:** Session persistence to retain multi-turn conversation history.
* **Data Portability:** One-click download tool to export chat logs as markdown files.
